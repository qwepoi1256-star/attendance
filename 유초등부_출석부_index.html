📲 출석부를 앱처럼 사용하세요!
홈 화면에 추가하면 브라우저 없이 앱처럼 바로 실행할 수 있어요. 오프라인에서도 사용 가능!
📲 홈 화면에 추가하는 방법
🍎 아이폰 · 아이패드 (Safari)
1️⃣ 하단 가운데 공유 버튼 □↑ 탭
2️⃣ 스크롤해서 "홈 화면에 추가" 선택
3️⃣ 우상단 "추가" 탭 → 완료!
📱 안드로이드 (Chrome)
1️⃣ 우상단 ⋮ 메뉴 탭
2️⃣ "앱 설치" 또는 "홈 화면에 추가" 선택
3️⃣ "설치" 탭 → 완료!
💻 PC (Chrome / Edge)
1️⃣ 주소창 오른쪽 ⊕ 설치 아이콘 클릭
2️⃣ "설치" 클릭 → 완료!
💡 설치 후 앱처럼 전체화면으로 실행돼요
🔗 Google Apps Script URL
아직 URL이 없으시면 아래 순서대로 진행해 주세요:

1️⃣ Google 스프레드시트 새로 만들기
2️⃣ 상단 메뉴 → 확장 프로그램 → Apps Script
3️⃣ 함께 받으신 Code.gs 파일 내용 전체 붙여넣기
4️⃣ 저장(💾) → 배포 → 새 배포
5️⃣ 유형: 웹 앱 / 액세스: 모든 사용자 선택 후 배포
6️⃣ 표시되는 URL을 아래에 붙여넣기
